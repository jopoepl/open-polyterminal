# Contributing

Thanks for helping improve PolyTerminal. We welcome issues, bug fixes, and UI polish.

## How to Contribute
1. Fork the repo and create a feature branch.
2. Keep changes focused and small.
3. Update documentation when behavior changes.
4. Open a PR with a clear description and screenshots for UI changes.

## Development
```bash
cd apps/terminal
npm install
npm run dev
```

## Code Style
- Keep components small and readable.
- Prefer plain functions over complex abstractions.
- Avoid adding new dependencies unless necessary.

## Reporting Issues
Please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots if relevant
